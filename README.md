# Personal Finance Management System

A desktop application developed for personal finance tracking, budget management, and financial analysis.

## Project Overview

This application helps users manage their income and expenses, create budgets, and monitor financial goals through reports and visual analytics.

## Technologies

- C#
- .NET
- MySQL
- Windows Forms

## Features

- Income tracking
- Expense tracking
- Budget management
- Financial goal creation
- Monthly reports
- Data visualization
- User-friendly interface

## Database

- MySQL

## Future Improvements

- Cloud synchronization
- Mobile application integration
- Export reports to PDF and Excel
- Multi-user support
